Summary

GE Pulseband is a wearable health monitor that provides users with real-time data such as blood pressure, heart rate, steps, and sleep. I created a web app to allow the user to monitor this data over time. The icons on the dashboard allows users to switch between the four different types of data and gives a graph that tracks the pattern of the selected data type in a given range. Basic HTML/CSS and JavaScript were used to show the interactions between the selections of different data types.
