Summary

I first layed out a 12 column grid based design for the app using photoshop. PHP was used to fetch data from the USGS database and switch between different rivers monitored by the Illinois Water Science Center. PHP was also allows the user change the date range of the graphs that display nitrate/phosphate data. The map was displayed through the use of the Esri API and displays the location of the selected rivers through JavaScript.
