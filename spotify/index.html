Summary

My partner and I decided to create a Jukebox Web App for Spotify where I would primarily focus on development. After reviewing the Spotify and Echo Nest API, we agreed upon the app’s features. The features we agreed upon were also divided into necessary and “nice to have”. This allowed us to first create an app with the necessary features and then focus on the “nice to have” features afterwards. Using a scrumboard, we devised two 5-day sprints that created daily tasks working towards completing each feature.

Development

I began by focusing on the search function of the app. The search function required an autocomplete function through JavaScript/jQuery. Using the suggest method in the Echo Nest API, I was able to obtain a list of recommended artists as a user was typing their desired artist.

Then, once the user selects an artist, another AJAX call was required for the data from the Spotify API which included the artist’s image, top tracks, albums, and related artists.
